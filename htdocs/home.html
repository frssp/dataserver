What is Zotero?

Zotero is a free, open-source reference manager that helps you collect, organize, annotate, and cite research. This server hosts your Zotero libraries internally, so your items and notes sync between the desktop app and the web — without leaving the corporate network.

Learn more about Zotero at zotero.org →

Getting started

1

Create an account

Register to get your username and an API key for syncing the desktop client and accessing the API.

 2

Connect the Zotero app

Point desktop Zotero at this server by setting extensions.zotero.api.url. Step-by-step instructions are in the guide.

 3

Browse in your browser

Open the Web Library to view, search, and export items, collections, and tags — no desktop app required.

 4

Create & share a group library

Set up a group library to collaborate — share references, notes, and collections with your team.

Don’t have the desktop app yet? Download Zotero → Then point it at this server (see the guide).
