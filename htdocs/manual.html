User Guide

How to use this self-hosted Zotero server — create an account, browse your library in the browser, and sync the Zotero desktop app.

Metadata-only sync. This server syncs your references, collections, tags, notes, and saved searches. It does not sync attached files or PDFs — those stay on the device where you added them.

1 Create an account

You need a username and password before you can sign in anywhere. Anyone with access to this server can self-register.

  1. Open the sign-up page and fill in a username, email, and password.
  2. Submit — your personal library and an API key are created automatically.
  3. You'll be signed straight into the Web Library.

Create your account →

Already registered? Just sign in to the Web Library with the same username and password.

2 Browse in the Web Library

The Web Library is a read-and-browse view of your references in the browser — no install required.

Open Web Library

What you can do

  • Three-panel layout — collections on the left, the item list in the middle, item details on the right.
  • Switch libraries — your personal library and any group libraries you belong to.
  • Filter & search — narrow by tag, or search title / creator / year.
  • Sort & choose columns — click a column header to sort; pick which columns are visible (saved in your browser).
  • Item details — Info, Notes, and Tags tabs for the selected item.
  • Export — right-click an item or collection to export to BibTeX, BibLaTeX, RIS, CSL JSON, or CSV.
The Web Library is for viewing and exporting. To add and edit references, use the Zotero desktop app (next section), which syncs with this server.

3 Sync the Zotero desktop app

Point the Zotero desktop app at this server so it syncs here instead of zotero.org. This is a one-time setup. Use Zotero 7 or newer.

Step A — Point Zotero at this server

  1. In Zotero, open Edit → Settings (on macOS: Zotero → Settings).
  2. Go to the Advanced tab → Config Editor, and accept the warning.
  3. Set these two preferences (use the search box to find each, double-click to edit):
PreferenceValue
extensions.zotero.api.url… (include the trailing slash)
extensions.zotero.streaming.enabledfalse
Trailing slash matters. The API URL must end in / or sync requests will fail.

Restart Zotero after changing these.

Step B — Sign in

  1. Open Settings → Sync.
  2. Click Sign In / Link Account. Zotero opens a login page in your browser.
  3. Enter the username and password you registered with, and confirm. The desktop app picks up your access automatically.
  4. Back in Sync, your account now shows as linked. Click Sync (the green circular-arrows icon in the toolbar) to sync.

Day-to-day

  • Add, edit, and organize references in the desktop app as usual.
  • Press the Sync button (or let auto-sync run) to push changes to the server.
  • On another computer, repeat this setup and sign in with the same account to get the same library.
Files don't sync. Attachments and PDFs are not uploaded. Keep "File Syncing" options in the Sync settings turned off — only library/metadata sync works here.

4 Group libraries

Group libraries let several people share one collection of references.

  • Groups you're a member of appear automatically — in the desktop app's left pane and in the Web Library's library switcher.
  • Whether you can edit a group's library depends on your role (member / admin / owner) and the group's settings.
  • To be added to a group, ask the group owner or an administrator — see the next section.

You can review the groups you belong to on your Account page.

5 API keys (advanced)

An API key lets scripts or third-party tools read and write your library through the Zotero Web API. You normally don't need to touch this — signing into the desktop app and Web Library handles keys for you.

  • View and create keys on your Account page, under API Keys.
  • Send the key with each request as a Zotero-API-Key header, e.g. GET /users/<id>/items.
  • Treat keys like passwords. Delete any key you no longer use.

FAQ

My PDFs aren't showing up on another computer.

That's expected — this server syncs metadata only. The reference entry syncs, but the attached file stays on the computer where you added it.

Sync fails right after setup.

Double-check extensions.zotero.api.url ends with a /, that streaming.enabled is false, and that you restarted Zotero. Make sure you can open this site in the same browser Zotero uses for sign-in.

I forgot my password.

There's no self-service reset. Ask an administrator to set a new password for you from the Admin panel.

Can I use the official zotero.org account?

No. This is a separate, self-hosted server. Register an account here; it's independent of zotero.org.

For administrators

Administrators manage the server from the Admin panel (protected by the super-user credentials), where you can:

  • Add, inspect, and delete users; reset user passwords.
  • Create and configure group libraries, and add/remove members.
  • Issue or revoke API keys for any user.
  • Check server status (user / group / item counts, schema version).

Self-registration is open by default, so end users can create their own accounts via the Sign Up page without admin involvement.
